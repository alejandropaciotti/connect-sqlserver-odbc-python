And write:

```shell
[FreeTDS]

Description=FreeTDS
Driver=/usr/lib/x86_64-linux-gnu/odbc/libtdsodbc.so
Setup=/usr/lib/odbc/libtdS.so
FileUsage=1
CPTimeout=5
CPResue=20
Threading=1
```
